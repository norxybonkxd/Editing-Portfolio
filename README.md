# Editing-Portfolio

A clean, responsive video editing portfolio website built with vanilla HTML, CSS, and JavaScript.

## Features

- **Hero section** with animated headline and call-to-action
- **About section** with stats and skills
- **Portfolio/Work section** with category filtering and project cards
- **Services section** showcasing offered services
- **Contact form** with client-side validation and submission feedback
- **Modal** preview on portfolio card click
- Fully **responsive** (mobile-first)
- Smooth scroll, scroll-reveal animations, and active nav highlighting

## Stack

- HTML5 · CSS3 (custom properties, grid, flexbox) · Vanilla JavaScript
- No build tools or dependencies required — open `index.html` directly in a browser

## Getting Started

```bash
# Clone the repo and open index.html
open index.html
```
